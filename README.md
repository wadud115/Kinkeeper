## 🚀 Project Name

**KeenKeeper**

---

## 📌 Short Description

KeenKeeper is a modern React-based friend relationship tracking application that helps users maintain meaningful connections. Users can track interactions like calls, texts, and video chats, visualize their communication patterns, and manage friendships in a simple, intuitive, and visually engaging way.

---

## ⚙️ Technologies Used

* React.js
* React Router
* Context API
* Tailwind CSS
* Recharts (for data visualization)
* React Toastify

---

## ✨ Key Features

### 🔹 Friend Interaction Tracking

Users can log interactions such as Call, Text, and Video. All activities are automatically stored and displayed in a centralized timeline.

### 🔹 Analytics Dashboard

An interactive donut (ring) pie chart provides a clear visual breakdown of communication types, helping users understand their interaction habits.

### 🔹 Instant Notifications

Real-time toast notifications give instant feedback whenever a new activity is added.

### 🔹 Friend Profile Management

Each friend has a dedicated profile page with detailed information, interaction history, and quick action buttons.
